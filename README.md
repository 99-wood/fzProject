# Project1 卷积和池化之基础版

1. 使用C语言（或伪代码）编写实现“一维卷积和池化”，对一维卷积后的结果进行池化操作。
2. 自己构建大于等于100个数据的一维数组，对其进行卷积和池化，卷积核长度需大于等于3，池化过滤器长度需大于等于3 。
3. 讨论是否有快速进行卷积和池化的编程实现方法。
